# KIN 7518: Assignment Resources & File Manifest

This document outlines the supplementary files and resources students will need to successfully complete the course assignments. These materials should be made available on Moodle or the course file repository.

## 1. General Course Resources
**[NEW] `GUIDE_TO_DATA_RESOURCES.md`**: **START HERE.** This document maps the "friendly" project names to the specific folders and reference readings in our course repository.
*   **`KIN7518_Syllabus_QIAN_UPDATED.pdf`**: The master syllabus document.
*   **`APA_7_Style_Guide_Quick_Reference.pdf`**: A cheat sheet for APA 7 formatting.
*   **`Group_Sign_Up_Sheet.xlsx`**: A shared spreadsheet for students to self-organize.
*   **`Group_Role_Definition_Form.docx`**: A template for formalizing group roles.
*   **`Ethical_Reflection_Checklist.pdf`**: Guiding checklist for privacy, bias, and harm reduction.
*   **`Form_Peer_Evaluation_Confidential.docx`**: Peer rating form.
*   **`Template_Group_Report_Fixed_Structure.docx`**: Pre-formatted report template.
*   **`Template_Presentation_Slides.pptx`**: Slide template.
