## Coding Instructions

### QUALITATIVE CODING GUIDE

### B50 Polarization Study — KIN 7518 TheMcGradys

### PURPOSE

**Goal:** Validate whether keyword-matched comments actually express the assigned theme. Identify false positives, nuance, and outrage intensity.

**Sample:** 15 comments per theme × 9 themes = 135 comments total. Each coder codes independently.

### CODING COLUMNS

**Valid Match?:** Does the comment actually express the assigned theme?
  Yes = clear match
  No = false positive (keyword used in different context)
  Partial = loosely related but not a strong example

**Actual Theme:** If Valid Match = No or Partial, write what theme the comment actually belongs to (use theme names from Column B, or write 'None').

**Sentiment:** Overall emotional tone of the comment:
  Pos = Positive (supportive, celebratory, hopeful)
  Neu = Neutral (factual, informational, ambiguous)
  Neg = Negative (critical, angry, hostile)

**Outrage Level:** Intensity of moral outrage expressed (0–3):
  0 = No outrage
  1 = Mild frustration or disappointment
  2 = Clear anger or condemnation
  3 = Extreme outrage, threats, or dehumanizing language

**Notes:** Any observations about language, context, sarcasm, or ambiguity that affect coding decisions.

### INTERCODER RELIABILITY

**Process:** Both coders code independently. Then compare. Mark Agreement = Yes if Valid Match AND Sentiment match. Discuss disagreements and resolve.

**Target:** Aim for ≥80% agreement on Valid Match and Sentiment before proceeding to analysis.
